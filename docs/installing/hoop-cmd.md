# Hoop Command Line

Connect to private infra-structure without the need of a VPN

```shell
hoop
Connect to private infra-structure without the need of a VPN.
https://hoop.dev/docs

Usage:
  hoop [command]

Available Commands:
  admin         Experimental admin commands
  connect       Connect to a remote resource
  exec          Execute a given input in a remote resource
  help          Help about any command
  login         Authenticate at Hoop
  search        Search for content in sessions
  start         Runs hoop local demo
  version       Show version

Flags:
      --debug        Turn on debugging
      --debug-grpc   Turn on debugging of gRPC (http2) if applicable
  -h, --help         help for hoop

Use "hoop [command] --help" for more information about a command.
```
