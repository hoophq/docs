# DLP

Redact sensitive content on the fly with [Cloud Data Loss Prevention](https://cloud.google.com/dlp).

### Redact Rules

- Customers documents numbers
- Credit card info;
- User e-mail
- IP addresses and DNS
- **[More ...](https://cloud.google.com/dlp/docs/infotypes-reference)**
